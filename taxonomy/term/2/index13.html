Mini-tutorial para instalar el Tkabber bajo Mandrake

Enviado por khudsa el 6 October, 2003 - 19:07.
Tagged:
  • Clientes
  • Tutoriales

Primero deberás bajar el cliente Tkabber. Se puede encontrar en aqui. Una vez bajado su instalación es sencilla. Descomprimes el fichero con el comando: $ tar zxvf tkabber-(versión)beta.tar.gz, entras en el directorio que se habrá creado y como root (utilizando el comando su) tecleas el comando # make. Con esto el programa se instalará en tu disco duro y creará los enlaces simbólicos necesarios para poder ejecutarlo desde cualquier terminal con sólo teclear tkabber.

Pero para que tkabber funcione necesitaras ademas 4 librerias más. Estas librerias son: tcl,tk, tcllib y bwidget.Para instalar las dos primeras solo hace falta hacer desde consola bajo usuario root: # urpmi tcl y # urpmi tk, con esto quedarán instaladas.Para instalar el tcllib simplemente (si tienes configurado el urpmi con repositorios externos) haz un urpmi tcllib, si esto no funciona te puedes descargar el rpm de cualquier página de busqueda de rpm's como rpmseek o de la ftp de rediris (rediris) e instalarlo como cualquier rpm. Para instalar el bwidget, una vez descargado (se puede encontrar en sourceforge), se descomprime y copias la carpeta que se habrá descomprimido donde este instalado el tcl. Para saber donde esta instalado haz un $ whereis tcl.

Pidgin (Gaim)

Enviado por natxo el 25 September, 2003 - 18:38.
Tagged:
  • Clientes

Pidgin (antes conocido como Gaim) es un cliente de Mensajería Instantánea para Linux, BSD, MacOSX y Windows. Es compatible con un montón de redes, entre ellas Jabber.

Si bien es un buen cliente para algunos protocolos, el soporte que tiene para Jabber es algo más limitado que el de los clientes específicos para Jabber, aunque un usuario de a pie seguramente no necesite más.

Un punto a tener en cuenta con Pidgin es que llama a Jabber "XMPP". XMPP es el nombre técnico de Jabber.
