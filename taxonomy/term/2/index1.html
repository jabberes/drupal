Pandion vuelve a la vida, y ahora es software libre

Enviado por naw el 21 July, 2009 - 16:57.
Tagged:
  • Clientes

Despues de mucho tiempo sin tener novedades acerca de Pandion, nos enteramos de que el desarrollador vuelve a tener tiempo para trabajar en el proyecto. Se va a proceder a una reescritura parcial del programa para obtener un cliente más rápido y estable. De momento, podemos ver que el sitio web ha sido rediseñado y que el código del proyecto ahora está disponible bajo una licencia de software libre: GPLv3.

Se ha lanzado Psi 0.13-rc1

Enviado por naw el 24 May, 2009 - 20:35.
Tagged:
  • Clientes
  • Lanzamientos

Hoy ha sido lanzada la primera Release Candidate de Psi 0.13. Esta versión soportara el plugin PsiMedia que permitirá mantener comunicaciones de voz usando Jingle RTP. De momento solo hay soporte bajo Mac OS X y Linux. Actualización: La RC2 ha añadido soporte para Windows.

Podéis descargároslo de Sourceforge. El fichero psi-0.13-rc1-linuxx86.tar.bz2 es una versión ya precompilada para linux.

He estado probándolo con Panko y hemos podido hablar. Si no os funciona, aseguraros de configurar un servidor STUN en la configuración de la cuenta. Podéis usar stunserver.org.

Lanzado Cocinella 0.96.12

Enviado por naw el 19 March, 2009 - 02:39.
Tagged:
  • Clientes
  • Lanzamientos

Se ha lanzado una nueva versión de Coccinella. Esta versión está dedicada a Mats Bengtsson, fundador de Coccinella, que falleció recientemente.

Los cambios de esta versión se mejoran la internacionalización (migrando a gettext las traducciones) y resuelven algún problema con los transportes.

Nueva versión de Coccinella

Enviado por naw el 25 September, 2008 - 09:08.
Tagged:
  • Clientes
  • Lanzamientos

Se acaba de anunciar el lanzamiento de la versión 0.96.10 del cliente Coccinella.

Esta nueva versión incluye mejoras visuales, como la inclusión del tema de iconos Oxigen, un panel de control que facilita el acceso a User Mood y User Activity. También se ha mejorado el dialogo de login y el tiempo de carga de la aplicación.
