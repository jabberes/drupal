Nueva versión de Psi: 0.9.3

Enviado por badlop el 12 January, 2005 - 20:20.
Tagged:
  • Clientes

Ha salido una nueva versión de Psi, el cliente Jabber multiplataforma.
No hay enormes cambios, pero sí un montón de pequeños añadidos. Entre otros:

  • Notificación de que el contacto te está escribiendo
  • Soporte x:data (para formularios de registro, etc)
  • Pequeña reorganización del diálogo para añadir nuevo usuario
  • Transferencia de ficheros con coger y arrastrar
  • Posibilidad de especificar un mensaje de estado al desconectar. Util por ejemplo si vas a estar desconectado varios dias; en tal caso te desconectas poniendo el mensaje de estado 'Me voy de viaje hasta marzo de 2005'. Tus contactos podrán ver este mensaje aunque estés desconectado.

Sale Pandion 2.1

Enviado por leandro@jabberes.org el 15 December, 2004 - 20:07.
Tagged:
  • Clientes
  • Lanzamientos

Acaba de salir la versión 2.1 de Pandion, el cliente Jabber para Windows.

Algunas de las mejoras:

  • soporte para moderación de salas de charla MUC.
  • también está muy curiosa la posibilidad de cambiar los fondos de la ventana de conversación. (A ver si alguien se anima y se curra unos fonditos chulos ...)
  • posibilidad de elegir el idioma de la interfaz
  • posibilidad de importar/exportar contactos en un fichero .xml
  • para los administradores, se cuenta ahora ademas con una consola de control para jabberd2.

Lanzamiento Psi 0.9.3-test1 y traducción al castellano

Enviado por bpeople@gentoo-... el 23 November, 2004 - 22:12.
Tagged:
  • Clientes
  • Lanzamientos

Ya ha salido, calentito del horno, una versión de prueba de Psi 0.9.3. En principio no lleva grandes cambios:

  • Notification when a user is typing
  • Advanced Data Forms capability (x:data)
  • Mac version uses the menu bar and also has a new color scheme
  • Slightly improved new user dialog
  • Drag and drop File Transfer
  • Ability to specify a status messages when going offline
  • The tray icon under X11 should now be more beautiful
  • Show self-contact is now a toolbar button instead of a normal option
  • There is now an option for all chat windows to be compact by default
  • Bugfix: file transfer sometimes wouldn't send the whole file
  • Fixed many other small bugs
  • Some usability improvements

Traducción de Psi 0.9.2 al castellano

Enviado por bpeople@gentoo-... el 6 September, 2004 - 21:32.
Tagged:
  • Clientes
  • General
  • Lanzamientos

Hace un par de meses salió la última versión de Psi (versión 0.9.2). Después de unos días, fueron apareciendo las diferentes traducciones a otros idiomas. Viendo que no estaba disponible y que en los foros de Psi no había actividad ninguna me decidí a hacerla.

El fichero de traducción está listo a falta de que el encargado de las traducciones lo suba junto con la página de descarga traducida al español. Mientras, si alguien está interesado en tener Psi 0.9.2 en castellano puede descargar el fichero desde aquí.
