Nueva versión de ejabberd: 0.9.8

Enviado por badlop el 3 August, 2005 - 08:27.
Tagged:
  • Lanzamientos
  • Servidores

Acaba de publicarse ejabberd 0.9.8. Algunas de las novedades son:

  • Los Virtual hosts (múltiples dominios) son más configurables: pueden tener diferentes métodos de autenticación, ACLs...
  • Módulo Publish-Subscribe actualizado.
  • API de desarrollo: nuevo hook filter_packet para filtrar o modificar paquetes al vuelo cuando fluyen por ejabberd.
  • Nuevo instalador gráfico para Linux/x86 que incluye las partes necesarias de Erlang/OTP (instalar y listo).

Nueva versión de Jive Messenger: 2.2.0 Beta 1

Enviado por peralta el 7 July, 2005 - 07:07.
Tagged:
  • Lanzamientos
  • Servidores

Ya tenemos la nueva beta de Jive Messenger, el servidor Jabber Open Source multiplataforma de la gente de Jive Software. Esta vez se trata de una revisión mayor, donde aparte del arreglo típico de fallos tenemos la característica estrella de esta versión: soporte de comunicación interservidor. Por alguna razón, esta característica no había sido considerada de vital importancia hasta ahora por sus desarrolladores, lo que impedía que ningún servidor Jabber público utilizase su software. Con las versiones 2.2 esto cambiará. Otra gran novedad es el soporte de componentes externos.

ejabberd 0.9.1 - nueva versión para corregir fallos

Enviado por peralta el 24 May, 2005 - 13:49.
Tagged:
  • Lanzamientos
  • Servidores

Se acaba de lanzar la última versión de ejabberd, el popular servidor Jabber multiplataforma escrito en Erlang (es el que se utiliza en JabberEs).

Se trata principalmente de una versión para corregir los fallos de ejabberd 0.9. Las pocas novedades adicionales son:

  • Mejora de los servicios de MUC e IRC (actualizar a la última versión del JEP)
  • Dos nuevas características en la administración via web: Tabla de la Última Actividad y Iniciar/Detener Módulos
  • Pequeña mejora del servicio de Publish/Subscribe
  • Traducciones actualizadas (inclutendo el castellano, claro)
