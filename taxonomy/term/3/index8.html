Enviado por peralta el 16 December, 2004 - 17:48.
Tagged:
  • Lanzamientos

Al hacer poco más de un mes de la último lanzamiento de Jabberd2, tenemos una nueva versión: Jabberd2-stable6. No se han introducido nuevas características pero sí que se han arreglado dos bugs importantes: uno permitía tumbar el servidor mandando un paquete de presencia a un usuario inexistente y el otro no permitía autenticar si se utilizaba el mecanismo de autenticación externa.

Nueva versión de jabberd2: 2.0s5

Enviado por peralta el 30 November, 2004 - 19:59.
Tagged:
  • Lanzamientos
  • Servidores

Hace un par de dias se anunció la liberación de una nueva versión de Jabberd2. Parece que últimamente se están poniendo las pilas con el desarrollo de este servidor, que estuvo a punto de caer en el abandono.

Como gran novedad, se incluye el soporte para certificados SSL encadenados (SSL certificate chaining) y para conexiones servidor a servidor (s2s) cifradas con SSL.

Nuevo Ejabberd 0.7.5

Enviado por badlop el 14 October, 2004 - 19:49.
Tagged:
  • Servidores

Ha salido nueva versión de
Ejabberd,
el servidor Jabber libre más potente.

Descarga para Linux y *BSD o instalador para Windows.
Tutorial de instalación en Linux.

Lista de mejoras

  • Soporte de STARTTLS en conexiones cliente2servidor (ahora solo falta que las implemente servidor2servidor) (1)
  • Soporte de autentificación mediante script externo
  • Nuevo módulo que implementa JUD y vCard usando LDAP
  • Mejoras en la administración via web (crear/borrar usuarios, añadir y quitar contactos de su lista, ver mensajes diferidos)
  • Soporte para announce/all y announce/motd (enviar mensaje a todos los conectados; poner mensaje del dia) (2)
  • Soporte para captura de mensajes usando una herramienta externa, como Bandersnatch (ya están trabajando para que no haga falta herramienta externa)
  • Soporte para expiración de mensaje (JEP-0023)
  • Soporte para búsqueda en JUD con prefijo
  • Añadido límite en el número de resultados en una búsqueda por JUD
  • Soporte de HTTPS en la interfaz web (3)

Ahora el servidor Jive Messenger es Open Source!!!

Enviado por gdombiak el 5 October, 2004 - 21:01.
Tagged:

El servidor XMPP Jive Messenger que anteriormente era comercial pasa a ser de ahora en más Open Source.
