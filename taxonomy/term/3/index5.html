Nueva versión del servidor Jive Messenger: 2.3.0

Enviado por gdombiak el 19 November, 2005 - 00:47.
Tagged:
  • Lanzamientos
  • Servidores

Ya tenemos la nueva versión de Jive Messenger, el servidor Jabber Open Source multiplataforma de la gente de Jive Software. Luego de contar con 2 versiones beta ahora podemos encontrar la versión final. La versión 2.3.0 se centró en:

  • Aumentar la compatibilidad con el protocolo XMPP
  • Las conexiones cliente-servidor ahora pueden ser aseguradas utilizando TLS
  • Los usuarios pueden autenticarse usando SASL.
  • Muchas mejoras fueron incluidas
  • y como regalo final ahora contamos con soporte para JEP-50: Ad-hoc Commands.

Nueva versión de jabberd2: 2.0s10

Enviado por peralta el 11 October, 2005 - 10:16.
Tagged:
  • Lanzamientos
  • Servidores

Hace unos días se liberó la última versión de la rama estable de Jabberd2, solucionando pequeños fallos y con un cambio interno estructural bastante importante. A partir de ahora, las estructuras de datos que se reservan por usuario en memoria ahora son dinámicas, lo que permite que el servidor escale mucho mejor bajo condiciones de carga elevada.

Nueva versión de Jive Messenger: 2.3.0 Alpha 1

Enviado por gdombiak el 14 September, 2005 - 14:23.
Tagged:
  • Lanzamientos
  • Servidores

Una nueva versión del servidor Open Source Jive Messenger ha salido.

La nueva versión incluye suporte de TLS (c2s), SASL (c2s) y autenticación PAM. El log completo de cambios puede ser encontrado aquí. Para obtener el servidor puedes hacerlo desde aquí.

Esta es una versión Alpha no porque el código este inmaduro sino porque se continuarán agregando nuevas funcionalidad a la versión final. Esta nueva versión ha sido testeada extensamente usando varios de los clientes más populares. En caso de encontrar fallas o querer aportar sugerencias de mejoras favor de hacerlo en los foros de soporte.
