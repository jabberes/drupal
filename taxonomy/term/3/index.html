Publicados ejabberd 2.1.9, 3.0.0-alpha-4 y exmpp 0.9.8

Enviado por naw el 4 October, 2011 - 18:48.
Tagged:
  • Lanzamientos
  • Servidores

Se acaba de publicar la versión 2.1.9 del servidor ejabberd.

Entre las mejoras que trae se incluyen:

  • Soporte de autenticación SCRAM-SHA-1
  • Nuevas opciones de LDAP
  • Soporte para entrar en salas simultaneamente compartiendo nick
  • Soporte de petición de voz en salas

Asimismo se han corregido bugs y realizado mejoras de rendimiento.

También se ha publicado una nueva alpha de ejabberd3 y una nueva versión de exmpp.

Lanzado Prosody 0.8

Enviado por naw el 7 April, 2011 - 15:19.
Tagged:
  • Lanzamientos
  • Servidores

Se acaba de presentar la versión 0.8 de Prosody. Prosody es un servidor Jabber/XMPP diseñado para ser ligero y fácil de usar.

Esta nueva versión incluye novedades como:

  • Plugins de autenticación. Permitiendo almacenar las contraseñas en claro o hasheadas. Además hay plugins en desarrollo que darían soporte de LDAP, autenticación de doble factor...
  • Plugins de almacenamiento (MySQL, PostgreSQL, SQLite3)
  • Comandos Ad-Hoc para administrar el servidor desde el cliente Jabber
  • Mejoras en el sistema de salas de charla

Felicidades al equipo de Prosody por la nueva versión.

Nuevas versiones de Openfire y Jabberd2, y pronto una de Prosody

Enviado por naw el 2 March, 2011 - 22:45.
Tagged:
  • Lanzamientos
  • Servidores

Acaba de presentarse la versión 3.7.0 del servidor Jabber/XMPP Openfire.

Entre los cambios podemos destacar que ahora usan una licencia Apache v2.0, mejor manejo de las conexiones inactivas, la aparición de un plugin de Jingle Nodes y correcciones de bugs varios.

Personalmente, me alegra ver una nueva versión de uno de los servidores más populares de Jabber/XMPP. Más aún cuando había visto especulaciones preocupantes sobre la vitalidad del proyecto.

Tambien merece la pena mencionar que, el pasado mes de febrero, jabberd2 presentó las versiones 2.2.12 y 2.2.13 con correcciones de fallos.

Y mientras tanto, Prosody, otro servidor, se acerca a la versión 0.8.0.

Se lanza la version 0.6 de Prosody (y 0.6.1)

Enviado por naw el 25 November, 2009 - 12:35.
Tagged:
  • Lanzamientos
  • Servidores

El equipo de Prosody ha lanzado la versión 0.6 de este servidor XMPP/Jabber y poco después, la versión 0.6.1 corrigiendo un problema en el roster.

Entre las novedades de esta versión se encuentran:

  • Compartición del nick al conectar desde distintos recursos de la misma cuenta
  • Compresión del trafico cliente-servidor
  • Cifrado de las conexiones entre servidores
  • Configuración del certificado SSL/TLS por cada Virtual Host
  • Soporte para importar bases de datos MySQL de ejabberd
  • Compatibilidad con el uso de VCards que hace ejabberd

Podéis descargarlo desde su página web.

Problemas entre google talk y las salas de charla

Enviado por naw el 30 January, 2009 - 13:47.
Tagged:
  • Servidores
  • Incidencias

Actualización: Google ha solucionado el problema y ya se puede acceder a las salas con normalidad usando cuentas de Google Talk.

Parece que el servidor de Google ha empezado a rechazar los mensajes de contactos que no están en la lista de contactos (roster) del usuario. Esto provoca que cuando una sala le envía mensajes, el servidor los rechaza devolviendo un error a la sala, y entonces esta expulsa al usuario.
