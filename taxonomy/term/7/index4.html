Nuevo ejabberd 1.1.2: corrección de errores, LDAP, MSSQL

Enviado por badlop el 6 October, 2006 - 13:38.
Tagged:
  • Lanzamientos
  • Servidores

Hay disponible una nueva versión del servidor libre ejabberd. Para más detalles consulta la noticia en la web de ejabberd. Principalmente incluye correcciones de errores y algunas mejoras:

  • Módulo LDAP mejorado
  • Posibilidad de limitar el número de sesiones abiertas por usuario.
  • Corregido un defecto importante en HTTP-poll que dejaba las sesiones abiertas permanentemente
  • Soporte de Microsoft SQL Server
  • Nuevo instalador para Windows que incluye Erlang

La lista completa de cambios está en ejabberd 1.1.2 release notes.

Nuevo ejabberd 1.1.1

Enviado por badlop el 26 April, 2006 - 15:33.
Tagged:
  • Lanzamientos
  • Servidores
Aviso: ejabberd 1.1.1 corrige un error introducido en ejabberd 1.1.0 acerca de autenticación SASL Anónima. Es muy recomendable actualizar de ejabberd 1.1.0 a 1.1.1.
Acaba de anunciarse la versión 1.1.1 del servidor Jabber libre ejabberd. Entre los cambios destacan:
  • Soporte para los siguientes JEPs: JEP-0050: Comandos Ad-Hoc, JEP-0138: Compresión del flujo de datos, JEP-0175: SASL anónimo.
  • Soporte nativo de MySQL.
  • Respecto a Multi-User Chat: Historial de conversaciones de las salas y otras mejoras.
  • Mejoras del rendimiento en varios apartados: TCP, logger, Erlang/OTP, y clustering.
  • Se puede limitar el tamaño de los mensajes XMPP. Esto es especialmente útil para limitar el tamaño de los avatares almacenados en las vCards.
La lista completa de cambios está en ejabberd 1.1.1 release notes.
