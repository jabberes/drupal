Conéctate desde detrás de un cortafuegos

Enviado por peralta el 22 May, 2004 - 20:08.
Tagged:
  • Lanzamientos
  • Servidores

Son muchos los posibles usuarios de Jabber que no pueden conectarse a este sistema de mensajería porque su única conexión a Internet se reduce a páginas web, estando los puertos del resto de servicios filtrados.

Jabber hace tiempo que tiene una solución para este problema y está definida en un [JEP]: Jabber HTTP Polling. Por esta razón nos hemos decidido a lanzar un servicio de este tipo, de forma que todos aquellos usuarios que viven detrás de un cortafuegos que restrinja el protocolo Jabber puedan conectar.

Para poder usar el servicio debemos utilizar un cliente que soporte este tipo de conexión, como pueden ser [Exodus] o [Tkabber]. Debemos especificar la URL http://poll.jabberes.org allí donde nos pida la URL para HTTP Polling (también se permite HTTPS). A partir de ahora, nuestro cliente conectará utilizando peticiones HTTP en vez de los puertos TCP 5222 o 5223.

Nuevo Smack 1.3.0

Enviado por gdombiak el 12 March, 2004 - 16:44.
Tagged:
  • Clientes
  • Lanzamientos

Smack es una librería Open Source para clientes XMPP que esta siendo usada tanto en dispositivos móviles como en grandes servidores. La nueva versión incorpora Service Discovery, un nuevo debugger y muchas otras mejoras.

  • Extremadamente fácil de usar pero al mismo momento con una poderosa API. Para enviar un mensaje no se necesitan más que 3 líneas de código:
    XMPPConnection con = new XMPPConnection("jabber.org");
con.login("mtucker", "password");
con.createChat("jsmith@jivesoftware.com")
    .sendMessage("Howdy!");
  • No hace falta programar al nivel de paquetes como otras librerías. Smack provee una interfaz para trabajar a alto nivel como por ejemplo las clases Chat y GroupChat, lo cual permite desarrollar de modo eficiente.

Alcanzado nuestro primer millar de usuarios registrados en Jabber

Enviado por badlop el 16 December, 2003 - 17:00.
Tagged:
  • Lanzamientos

Bueno, el servidor Jabber JabberES.org tiene ya más de mil cuentas registradas desde que fue lanzado a finales de septiembre.

Aunque la mayoría de las cuentas creadas no se usan, actualmente suele haber unos cincuenta usuarios concurrentes. Es decir, un 5%, que concuerda con lo indicado en la guia para el administrador de un servidor Jabber (the number of concurrent users will typically average less than 5% of registered users for a service provider).

Por cierto, el número de usuarios registrados en la web asciende a unos quinientos, aunque probablemente muchos están duplicados, pues acceden tanto con la cuenta Drupal como con la cuenta Jabber.
