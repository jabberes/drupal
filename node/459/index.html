Acabo de hacer un post sobre ello.

Yo uso dos pcs con jabber las dos, a una le doy prioridad 4 y a la otra le doy prioridad 5 (creo que la 5 es la que viene por default en todos los clientes, corrijanmen si me equivoco), ahora, cuando los dos clientes esten prendidos los mensajes (TODOS los que se envien desde que se inicio sesion con el de prioridad 5) se enviaran al de priority mas alta.
Y cuando deslogee el de prioridad 5 automaticamente todos los mensajes van a parar a el de prioridad 4.

Esta bien esto, asi como lo plantee?

Correcto, por defecto iran al de mayor pioridad. El que traigan 5 por defecto depende del cliente.

1)Prioridades, no importa el valor que le asigne? es solo para utilizar multiples clientes y no afecta en nada el numero que le asigne o esto cambia en algo con mi manera de proceder con el servidor?

Puees asignar entre -128 y 127. Pero si no entiendo mal lo de abajo, si usas negativas no podrás recibir.

2)Si tengo con los dos clientes la misma prioridad, que pasa?

El RFC3921 dice:
If two or more available resources have the same priority, the server MAY use some other rule (e.g., most recent connect time, most recent activity time, or highest availability as determined by some hierarchy of values) to choose between them or MAY deliver the message to all such resources. However, the server MUST NOT deliver the stanza to an available resource with a negative priority; if the only available resource has a negative priority, the server SHOULD handle the message as if there were no available resources (defined below)
O sea, que el servidor haga lo que crea conveniente, pero que no mande mensajes a prioridades negativas.

3)Si estoy usando transporte, eg: msn... que pasa con las prioridades, es lo mismo? todo lo que se envia va a la prioridad mas alta?

Creo que si.

4)Si en el punto uno me dicen que si influyen los numeros que le asigno, que numero hace cada cosa, y cuales son todos los numeros que se pueden asignar y que hace cada uno?

Ya te respondí arriba

5)Que prioridad usa el cliente jabbernaut, ya que no permite setearla?

La primera vez que oigo ese nombre......si el cliente no especifica una al prioridad se toma prioridad 0.
