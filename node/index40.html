JabberEs - mensajería instantánea libre

  • Con Jabber puedes enviar mensajes a usuarios desconectados, conectar a tu cuenta desde varios sitios al mismo tiempo, conectar a otras redes (como MSN, AIM o Yahoo!), poner tu estado en una web y mucho más...
  • Los servidores Jabber de todo el mundo conforman una federación de mensajería instantánea en la que todo el mundo puede hablar entre sí sin restricción alguna.
  • Jabber está basado en XMPP, un protocolo extensible, abierto y estándar basado en XML para el intercambio en tiempo real de mensajes y presencia entre dos puntos en Internet.
  • JabberEs ofrece información sobre Jabber en castellano, y diversos servicios gratuitos incluyendo un servidor Jabber público y un servicio de salas de charla.
Para comenzar, puedes crear una cuenta Jabber en tres sencillos pasos. Si tienes problemas o dudas que no estén resueltas en esta web, puedes probar a entrar en nuestra sala, bien desde tu cliente jabber, o vía web.

Jabber Journal #19 :: 2004-11-29

Bueno, como viene siendo costumbre pido perdón por no cumplir mi promesa de publicar el Noticiario Jabber más a menudo -- la última tirada salió hace 5 meses.

Pero también como es usual, la comunidad Jabber ha continuado activa. Entre otras cosas, la IETF finalmente publicó el XMPP RFCs
a primeros de Octubre, lo que conduce a un mayor interés en las tecnologías Jabber
para corporaciones, gobiernos y proyectos de código abierto.
Trabajamos en la aprobación de la IETF durante dos años y medio y estoy seguro de que algunos compañeros creen que no valió la pena el tiempo y el esfuerzo, pero yo creo que empezaremos a ver cada vez más beneficios por aquel tiempo gastado. XMPP (así es el nombre de la IETF para el núcleo de los protocolos Jabber) está subiendo en toda clase de aplicaciones, desde el Ejército americano hasta la integración de librerías públicas.
Cada vez más gente contacta conmigo cada día para comentar proyectos basados en Jabber en los que están trabajando. Desafortunadamente, no siempre puedo hablar de esos proyectos en un foro público, pero puedo asegurar que bastantes serán lanzados en los próximos 6 a 9 meses, así que tenemos un montón de ganas de que salgan.

Nueva versión de jabberd2: 2.0s5

Enviado por peralta el 30 November, 2004 - 19:59.
Tagged:
  • Lanzamientos
  • Servidores

Hace un par de dias se anunció la liberación de una nueva versión de Jabberd2. Parece que últimamente se están poniendo las pilas con el desarrollo de este servidor, que estuvo a punto de caer en el abandono.

Como gran novedad, se incluye el soporte para certificados SSL encadenados (SSL certificate chaining) y para conexiones servidor a servidor (s2s) cifradas con SSL.

La lista completa de correcciones y el paquete con los fuentes los teneis aquí.

¡Recomiéndanos tus recetas!

Enviado por peralta el 28 November, 2004 - 17:55.

A partir de la próxima semana, el mensaje del día de nuestro servidor Jabber (el mensaje que se recibe nada más conectarse) se irá actualizando regularmente. En la actualidad, no le estamos dando ningún uso a esta característica del servidor, así que hemos pensado que sería ideal dar pequeños consejos de uso de lo más variado.

Por ejemplo, podemos recordar la existencia de eduardo, o darte consejos para conectarte desde detrás de un cortafuegos, o pequeños trucos para aprovechar más determinados clientes, ... Ya tenemos alguna que otra receta pensada, pero seguro que nos dejamos un montón.

Lanzamiento Psi 0.9.3-test1 y traducción al castellano

Enviado por bpeople@gentoo-... el 23 November, 2004 - 22:12.
Tagged:
  • Clientes
  • Lanzamientos

Ya ha salido, calentito del horno, una versión de prueba de Psi 0.9.3. En principio no lleva grandes cambios:

  • Notification when a user is typing
  • Advanced Data Forms capability (x:data)
  • Mac version uses the menu bar and also has a new color scheme
  • Slightly improved new user dialog
  • Drag and drop File Transfer
  • Ability to specify a status messages when going offline
  • The tray icon under X11 should now be more beautiful
  • Show self-contact is now a toolbar button instead of a normal option
  • There is now an option for all chat windows to be compact by default
  • Bugfix: file transfer sometimes wouldn't send the whole file
  • Fixed many other small bugs
  • Some usability improvements
