Publicación de Psi 0.11 RC1

Contenido archivado

El contenido de la web se encuentra archivado y no se podrá crear nuevo contenido. Más información.

Enviado por naw el 12 May, 2007 - 12:58.
Tagged:
  • Clientes
  • Lanzamientos

Psi se acerca a la nueva versión, tal y como podemos ver. Esta Release Candidate está destinada a que la gente busque fallos en ella y a que los traductores empiecen a traducirla.
Source: http://downloads.sourceforge.net/psi/psi-0.11-RC1.tar.bz2
Mac: http://downloads.sourceforge.net/psi/Psi-0.11-RC1.dmg
Windows: http://downloads.sourceforge.net/psi/Psi-0.11-RC1-win32.zip

Lista de cambios:

  • Soporte para salas de charla multiusuario XEP-0045
  • Soporte para almacenar marcadores, XEP-0048, para autoconectar a salas. De momento no se pueden crear con Psi, puedes usar Gajim o Tkabber para ello
  • Accesos directos personalizables
  • Corrección ortográfica
  • Psi ahora usa QT4 (4.2.3) y la librería QCA2
  • Soporte para listas de privacidad permitiendo bloqueo avanzado de usuarios
  • Permitir usar el nombre del host como recurso
  • Nuevos iconos
  • Soporte de comandos AdHoc XEP-0050
  • Soporte preliminar para el Protocolo de Eventos Personales (PEP, una simplificación de PubSub) XEP-0163
  • Soporte preliminar de control remoto de clientes XEP-0146 (permite que, al estar conectado desde dos sitios distintos puedas recoger los mensajes recibidos en el otro cliente, o cambiar el estado)
  • Soporte para el XEP-0138 (Compresión de tráfico)
  • Reprogramado el sistema de creación de cuentas (incluyendo soporte para el XEP-0077)
  • Soporte de formularios en mensajes (XEP-0004)
  • Soporte para el XEP-0070 (Verificación de peticiones HTTP a través de XMPP)
  • Soporte para el XEP-0033 para usar multicasting al enviar mensajes a varios usuarios (Usar option.enable-multicasting para activarlo)
  • Mejor manejo de errores, incluyendo XEP-0086
  • Soporte para cambiar la prioridad sin necesidad de desconectar
  • Mejores mensajes de error con PGP
  • Esconder el selector del obsoleto protocolo para Service-Discovery
  • Nuevo sistema de opciones en el archivo options.xml
  • Soporte para avatares guardados en la VCard XEP-0153
  • Soporte preliminar para apodos de usuario XEP-0172
  • Soporte preliminar para intercambio de elementos de la lista de contactos XEP-0144
  • Reestructuración de gran parte del código
  • Solucionado un bug que no enseñaba los cambios de asunto en salas
  • Ejecutables Universales en MacOS
  • Soporte para notificación de estado de charla XEP-0085
  • Soporte para reconocer las características de una entidad (reduce el ancho de banda) XEP-0115
  • Existe una opción en el config.xml para para usar el tema brushed metal en MacOS en algunas ventanas
  • Translucidez de ventana para las tres platafornas
  • Corrección de un fallo de direcciones Unicide
  • Autoreconectar al salir de suspensión
  • Soporte para copiar y pegar texto con emoticonos
  • Corrección de un bug que escondía emoticonos al responder y citar
  • El historial se construye más rápido que antes
  • Configuración de barras de herramientas desde las opciones
  • Ayuda contextual movible
  • Soporte para el compilador Ming32
