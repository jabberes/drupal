XMPP Roundup #11

Tagged:

[Informadores: Nicolas Vérité y Peter Saint-Andre]

Bienvenidos a la onceava edición del XMPP roundup, las noticias veraniegas de la galaxia XMPP.

Software nuevo y actualizado

Blather
Blather es una biblioteca para Ruby, bajo licencia BSD.

dojox

dojox, es una biblioteca XMPP para JavaScript que forma parte del toolkit Dojo.

Eiffel

Eiffel XMPP es una biblioteca para el lenguaje Eiffel, inspirado en XMPPHP y publicado bajo la Eiffel Forum License v2.

eM Client

eM Client es un cliente PIM (Personal Information Manager, que incluye email, calendario, contactos, etc) para Windows (XP y Vista) que integra XMPP.

glu

glu es un cliente XMPP para Windows, escrito en el framework .Net usando la biblioteca agsxmpp, licenciado bajo la licencia GPLv2

hxmpp

hxmpp es una biblioteca para el lenguaje haXe para clientes y componentes (sin licencia, por ahora)

SworIM

SworIM es un cliente XMPP nativo para iPhone.

Pandion

Pandion, el cliente XMPP, se ha liberado como software de código abierto bajo licencia GPLv3.

Prosody

Se ha lanzado la versión 0.5 del servidor Prosody

Servicios, nuevos o actualizados, basados en XMPP

Jake

Jake es un sistema colaborativo de intercambio de ficheros basado en XMPP.

PyGo Wave Server

PyGo Wave Server está disponible.

Wave en ejabberd

ProccesOne ha publicado un articulo sobre como configurar ejabberd para la implementación de referencia de Wave.

Collecta

Collecta es un motor de busqueda en tiempo real.

superfeedr.com

superfeedr.com es un sistema de parseo de feeds en tiempo real, que usa PubSub para notificaciones.

Herramienta de pruebas Weavver

La herramienta de pruebas Weavver es una herramienta XMPP para comprobar los registros SRV [un tipo de registros DNS que permiten que el servidor jabber alojado en example.com se encuentre en una IP distinta a la que apunta example.com] y puertos TCP.

Juick

juick.com es una red social y un servicio de microblogging basado en Jabber/XMPP que permite publicar avisos, canciones, fotos, geolocalización, estado de animo, vCard, etc.

Especificaciones XMPP nuevas y actualizadas

La XMPP Standards Foundation ha pasado el XEP-0198: Stream Management y el XEP-0249: Direct MUC Invitations de Experimental a Draft en su proceso de estandarización. Además, las siguientes especificaciones han sido acualizadas:

  • XEP-0051: Connection Transfer: version 0.2
  • XEP-0220: Server Dialback: version 0.4
  • XEP-0258: Security Labels in XMPP: version 0.4
  • XEP-0260: Jingle SOCKS5 Bytestreams Transport Method: version 0.3
  • XEP-0271: XMPP Nodes: version 0.1

Conclusión

No hay descanso en el entorno XMPP: incluso en verano las cosas van a un ritmo rápido.
