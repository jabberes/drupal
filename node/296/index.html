Una posible solución a "No acepta registro abierto"

Enviado por Anónimo el 28 September, 2004 - 21:50.

Mi problema era el mismo que el tuyo, y durante varias horas no supe que hacer. Después me dí cuenta que el error que devolvía el servidor era el 409 ( Usuario YA existente ) por lo que procedí a ingresar normalmente y.. voilá.. Ya tuve entrada. Aparentemente el Exodus o el Ejabberd tienen una forma poco elegante de confirmar el correcto registro de un nuevo nick: simplemente se quedan colgados. Pero al ingresar de nuevo, el nick ya está registrado y es posible habilitarlo como Admin y todo lo demás.
