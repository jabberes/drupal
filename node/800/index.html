Propuesta de E-Mail para promocionar el uso de Jabber (Wiki)

La idea de este post es crear un correo modelo (actualizado) para crear un email para promocionar el uso de jabber.

¿Imaginas un mundo en el que no pudieras mandar un email a alguien de Yahoo! sin tener cuenta con ellos? ¿Un mundo en el que para llamar a un teléfono móvil de Movistar tuvieras que tener contrato con Movistar?

¡Pues eso pasa con la mensajería instantánea! Para hablar con la gente de MSN tienes que tener una cuenta con ellos y para hablar con los de AIM, tienes que estar dado de alta con ellos. Alguien de Yahoo! no puede hablar con otra persona que use otro servicio distinto. Esto sucede porque los proveedores bloquean el acceso a terceras personas para retener a los usuarios en sus productos ¿No sería genial que existiera una alternativa? Existe, es Jabber.

Jabber es una red distinta, ha sido pensada para ser abierta, basada en estandares aprobados por el mismisimo IETF (la organización que especifica estándares tales como el HTTP que usamos para navegar) Esta filosofía permite que cualquiera pueda usar el programa que quiera, e incluso hacerse el mismo uno, para conectarse. También permite que cualquiera se monte su propio servidor. O que, alguien con cuanta en un servidor Jabber hable con gente y use los servicios de otro servidor Jabber.

Jabber tiene muchas posibiidades de ampliación, las cuales le han permitido incorporar características como:

  • Salas de charla, donde conocer a otra gente que usa Jabber
  • Mensajes a usuarios desconectados. Con Jabber puedes enviar un mensaje a alguien que no está conectado para que lo lea cuando conecte.
  • Pasarelas a otras Redes, podras hablar con los usuarios de MSN, Yahoo!, AIM y otros servicios de mensajería instantánea

Si te has decidido a probarlo y te preguntas como hacerte una cuenta, estás de suerte, es muy sencillo. Lo primero es que si usas GMail o Gizmo ya tienes una. Google Talk y Gizmo permiten usar Jabber, si bien, las posibilidades que te dan estos programas no son tam amplias como las de los que han sido diseñados para Jabber. Pero ¡no te preocupes!, ¡puedes usar estas cuentas con clientes especificos para Jabber! Si usas GMail tu Jabber-ID (JID) o Identificador Jabber sera usuario@gmail.com y si usas Gizmo: usuario@chat.gizmoproject.com.

Si no usas GMail o Gizmo peudes abrirte una una cuenta en cualquiera de los más de 150 servidores Jabber funcionando en todo el mundo. Para empezar puedes probar el servidor jabberes.org En este servidor tambien encontrarás información sobre el mundo Jabber.

Para conectarte deberás usar un programa. Si actualmente usas Google Talk, puedes usarlo. Gaim o Kopete también soportan Jabber. Pero, si no sabes por donde empezar, te sugiero Pandión, si usas Windows, o iChat si usas Apple. Tanto los usuarios de Windows como los de Apple o los de GNU/Linuix también puedes usar Psi.

Aparte de la interoperabilidad de Jabber, hay más razones para usarlo.

La red Jabber es mucho más resistente. En redes como MSN o AIM, las conexiones se realizan a un servidor central. Si este cae, todo el servicio cae a nivel mundial. En Jabber, al haber decenas de servidores, si uno cae, el resto de la red sigue funcionando y solo se ven afectados los usuarios de ese servidor. Además, si estos usuarios tuvieran una segunda cuenta en otro servidor, no se verían afectados.

Existen bots de estado que puedes usar para poner tu estado en tu sitio web.

¿Sabias que los mensajes que envías por protocolos como MSN se pueden espiar? Ahá, cualquiera que pueda acceder al tráfico que transmite tu ordenador, puede leer tus mensajes. Este puede ser una persona malintencionada, tu Proveedir de Acceso a Internet, o el proveedor de Mensajería instantánea. A este respecto, Microsoft en el contrato de MSN http://messenger.msn.es/Help/Terms.aspx?mkt=es-es establece que:

"Microsoft consideramos que el uso que el Usuario realice del Servicio, incluido el contenido de sus comunicaciones, es privado. De forma rutinaria, Microsoft no supervisará dichas comunicaciones ni revelará información sobre ellas a ningún tercero. Sin embargo, Microsoft podrá hacerlo, incluyendo el contenido de las mismas, si lo estima necesario para: (1) cumplir la legislación o para responder a un proceso legal; (2) garantizar el cumplimiento de este contrato por parte del Usuario o (3) proteger los derechos, propiedades o intereses de Microsoft, sus empleados, sus clientes o el público en general."

En Jabber esto se soluciona cifrando el tráfico que se envía al servidor. Además permite usar GnuPG de forma que ni siquiera el servidor pueda acceder a la conversación.

Tambien tienes la posibilidad de conectarte desde varios sitios a la misma cuenta. Es decir, dejar un programa conectado en casa, y conectarte a la cuenta desde el trabajo. Y la mayoría de los clientes Jabber permiten acceder a varias cuentas Jabber a la vez.

Algunos sitios interesantes que puedes visitar sobre Jabber son:

http://www.jabberes.org Una de las principales comunidades Jabber hispanas. En ella encontarás mucha información.
http://www.jabberland.com/ Esta es otra página sobre jabber. Ofrece un buscador de contactos y salas de charla, entre otras cosas. En http://amessage.de también puedes encontrar un buscador de contactos.
http://www.jobble.org/ permite que los usuarios de Jabber se puedan añadir en un mapa del mundo mostrando su estado de forma que, se pueda ver la distribución geográfica de los usuarios.

Para este post se han tomado ideas de:
http://www.deepdarc.com
http://n.ethz.ch/student/fbianco/odd/odd.html
http://www.jabber.org.au/
y, por supuesto, de http://jabberes.org.

Bonita idea, yo añadiria 3 cosillas (por ahora), como ideas para redactar el supuesto email:

- La gente pasa de leer: pon mas de 5 lineas y pasaran de mirarlo.
- La gente necesita motivos que ELLOS entiendan, para cambiar. Que es libre y bueno no les interesa, que tiene salas, por ejemplo, si.
- Y bueno, adicionalmente, necesitan que les digas:
"Toma, usa ESTE PROGRAMA, HAZTE LA CUENTA AQUI, y conectate ASI", vamos, instrucciones precisas, con un cliente concreto (a poder ser bonito), y con todo paso a paso.

Yo hice un tutorial de Psi en EOL precisamente por esto ultimo.

A la gente (generalizando, como he hecho en todo el post), le lia que le digas que hay muchos programas muy buenos, para elegir. Creo que es mejor que elijas por ellos, les digas, "usa este", y les guies para hacerlo (por facil que sea). Luego, mas adelante, ya probaran otros, si quieren.

Obviamente hablo del tipico usuario de MSN que todos conocemos. Al que es informatico o tiene habilidad con el ordenador, no hara falta que le guies tanto o le expliques tanto claro xD
