--with-extra-library-path=/usr/local/lib:/usr/lib/mysql

me funcionó correctamente para el mysql con fedora core 2. Por lo menos ya he podido pasar del script de configuración. Lo había intentado con --libdir=DIR pero al parecer esto no lo solucionó.
