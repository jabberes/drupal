O Bandersnatch o, como mucho J-EAI

Enviado por badlop el 3 May, 2005 - 14:33.

Pues a parte de una herramienta específica para esta tarea, como es Bandersnatch, podrías usar J-EAI, que es ejabberd con ciertos cambios. Entre ellos, la capacidad de ver los mensajes transferidos por los usuarios desde el administrador via web. Puedes probar J-EAI bajandote REPOS, que es un CD que contiene entre otros programas J-EAI ya compilado para Linux, Windows y Mac.

Eso sí, no me preguntes si se puede usar tu base de datos de ejabberd actual con J-EAI, ni como instalarlo, ni si soporta tal o cual, ni si tiene errores o no, porque no se absolutamente nada de él.
