A ver si aclaramos las

Enviado por badlop el 6 December, 2005 - 17:29.

A ver si aclaramos las cosas: ahora has iniciado el servidor con el acceso directo.

  1. ¿Pueden conectar los clientes?
  2. Al intentar entrar en la administración via web, a la que deberías poder entrar con la dirección http://localhost:5280/admin/, el navegador te muestra un mensaje de error. Por lo visto la página no está disponible. El servidor Jabber no está iniciado correctamente. O bien no lo has iniciado, o bien está iniciado pero con errores.
  3. Prueba lo siguiente: renombra el directorio 'database' a 'database-antigua' y luego inicia ejabberd. De esta forma ejabberd creará una base de datos nueva. Prueba a conectar ahora. De esta forma sabrás si el problema está en que la BD anterior ha quedado corrompida.
