JabberEs - mensajería instantánea libre

  • Con Jabber puedes enviar mensajes a usuarios desconectados, conectar a tu cuenta desde varios sitios al mismo tiempo, conectar a otras redes (como MSN, AIM o Yahoo!), poner tu estado en una web y mucho más...
  • Los servidores Jabber de todo el mundo conforman una federación de mensajería instantánea en la que todo el mundo puede hablar entre sí sin restricción alguna.
  • Jabber está basado en XMPP, un protocolo extensible, abierto y estándar basado en XML para el intercambio en tiempo real de mensajes y presencia entre dos puntos en Internet.
  • JabberEs ofrece información sobre Jabber en castellano, y diversos servicios gratuitos incluyendo un servidor Jabber público y un servicio de salas de charla.
Para comenzar, puedes crear una cuenta Jabber en tres sencillos pasos. Si tienes problemas o dudas que no estén resueltas en esta web, puedes probar a entrar en nuestra sala, bien desde tu cliente jabber, o vía web.

Conéctate desde detrás de un cortafuegos

Enviado por peralta el 22 May, 2004 - 20:08.
Tagged:
  • Lanzamientos
  • Servidores

Son muchos los posibles usuarios de Jabber que no pueden conectarse a este sistema de mensajería porque su única conexión a Internet se reduce a páginas web, estando los puertos del resto de servicios filtrados.

Jabber hace tiempo que tiene una solución para este problema y está definida en un [JEP]: Jabber HTTP Polling. Por esta razón nos hemos decidido a lanzar un servicio de este tipo, de forma que todos aquellos usuarios que viven detrás de un cortafuegos que restrinja el protocolo Jabber puedan conectar.

Para poder usar el servicio debemos utilizar un cliente que soporte este tipo de conexión, como pueden ser [Exodus] o [Tkabber]. Debemos especificar la URL http://poll.jabberes.org allí donde nos pida la URL para HTTP Polling (también se permite HTTPS). A partir de ahora, nuestro cliente conectará utilizando peticiones HTTP en vez de los puertos TCP 5222 o 5223.

Descubierto nuevo fallo de seguridad en MSN Messenger

Enviado por rrey el 5 May, 2004 - 11:16.
Tagged:
  • Usos y Abusos

Se ha descubierto una nueva vulnerabilidad en el software MSN Messenger de Microsoft que permite al atacante leer archivos arbitrariamente.

El problema afecta a las versiones 6.0 y 6.1 del MSN Messenger y está solucionado en la última versión 6.2. ¿Por qué siempre los usuarios de Messenger se enteran de estos problemas después de que haya salido una nueva versión de este software? ¿Es casualidad o lo usan para forzar la actualización?

Mientras tanto, si usas otro sistema operativo diferente a los soportados por el cliente de Microsoft, puedes estar tranquilo.
