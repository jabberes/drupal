Web Presence: indicador web de tu estado en Jabber

Hemos instalado en el servidor de JabberES un nuevo servicio: WebPresence.
Es un módulo de ejabberd que permite a cualquier usuario Jabber de jabberes.org mostrar su estado en la web.

La idea es similar a Eduardo, otro servicio que tubimos, pero que no era capaz de cumplir su cometido. Esperamos que Web Presence sí sea capaz :)

Para empezar con WebPresence hay dos pasos: primero registrarse en el servicio para permitir las opciones deseadas. Luego hay que averiguar qué direcciones web se pueden usar. A continuación explico con más detalle cada paso.

Primero te registras en el servicio Jabber con tu cliente Jabber. Por ejemplo, con Psi vamos a Descubrir servicios y hacemos doble click en Web Presence. Entonces veremos un formulario de registro que nos preguntará los tipos de url que queremos tener disponible:

  • Jabber ID: Deberemos especificar el JabberID en la dirección (ejemplo: http://jabberes.org:5280/presence/jid/usuario/jabberes.org/image
  • Random ID: Si no queremos que nuestro JabberID sea visible, podemos especificar un identificador en la dirección (ejemplo: http://jabberes.org:5280/presence/rid/12345678901234567890/image

A continuación, nos pregunta el tema de iconos que queremos usar por defecto y nos da la opción de tener disponible más información sobre nosotros.

Podemos hacer disponible nuestro avatar (o imagen de usuario), obtener la información en formato XML o en Javascript para embeberlo en nuestra web.

Segundo paso: ahora que ya hemos configurado el servicio Web Presence, vamos a usarlo convenientemente. Para poner tu imagen en una página web basta con que insertes el siguiente trozo html:
<img src="http://jabberes.org:5280/presence/jid/usuario/jabberes.org/image" />

También puedes usar esta web de ayuda para construir tus URLs.
