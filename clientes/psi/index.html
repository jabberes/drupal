Psi es un cliente Jabber libre, gratuito, multiplataforma (Windows, Linux y MacOSX). Tiene soporte de cifrado GPG.

  • Página oficial de Psi
  • Psi en la Wikipedia
  • Psi-psz, customized Psi Jabber Client

Psi+ es una versión alternativa de Psi que incluye características experimentales adicionales con la intención de probarlas, mejorarlas y finalmente incluirlas en el Psi general.

Tutoriales

  • Instalación Psi y creación de una cuenta, por Omarete.
  • GPG en Psi, por Bulma.

Cómo empezar con Psi

  • Lo primero es descargar el paquete con el programa. Se puede encontrar en la Web de Psi. No es difícil elegir el paquete que necesitas para tu plataforma:
    • Si usas Windows deberias descargar el Windows 9x/2000/XP Installer (with QSSL); El proceso de instalación es el corriente para aplicaciones de windows, y es suficientemente autoexplicativo.
    • Si usas Mac deberas descargar el Apple Macintosh (with QSSL). Sólo funcionará si usas MacOSX. El proceso de instalación es el habitual.
    • Si por contra eres un fan del sistema operativo del pingüino, deberás descargar y compilar el código fuente, o utilizar el método estandard de tu distribución.
  • Una vez descargado e instalado, hay que castellanizarlo. Para ello hay unas bonitas instrucciones en la web de Psi.
    • Si prefieres catalanizarlo tienes instrucciones también en ésta página
  • De acuerdo, ya lo tenemos instalado y castellanizado. Ahora toca configurarlo. Nada mas ejecutarlo nos aparece una ventana parecida a la siguiente, en la cual hay que seleccionar idioma, y crear un nuevo perfil.

  • Una vez creado el nuevo perfil, hay que configurar nuestra cuenta Jabber. Sólo hay que rellenar las casillas con nuestros datos.

  • Por último damos a conectar, y ¡¡ya tenemos el cliente de Jabber como diox manda!!

http-poll en psi

Enviado por badlop el 2 November, 2004 - 19:23.

A mi con Psi CVS me funciona, comprueba las opciones:

  • Conexión -> Avanzado -> Usar SSL: desactivado
  • Conexión -> Proxy -> Editar:
    • Tipo: HTTP Polling
    • Host: vacio
    • Port: 0
    • Polling URL: http://jabberes.org:5280/http-poll
    • Use autenthication: desactivado

Recuerda abrir el puerto 5280 en el cortafuegos.
